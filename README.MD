## Instalación
- MySQL: Como prerequisito, el equipo donde corra el sistema, debera contar con acceso a una instancia de MySQL.
- Inicializar la base de datos: Ejecutar el script `.sql` para inicializar la base de datos: [Proyecto.sql](./Datos//Proyecto.sql).
- Ejecutar la aplicacion desde el `.exe` inclido en la carpeta `Release` o desde Visual Studio.
